@interface EnforcerCompileStatic

      The annotation will statically compile a method or class, with the method taking precedence, however it won't interfere with enforce based annotations, like the traditional

      CompileStatic:
      will. This annotation takes the same parameters as
      CompileStatic:
      as it uses the same transform under the covers. parameters TypeCheckingMode the type checking mode pass or skip. extensions any type extensions you would like to add, by default this annotation adds the same extensions as
      @GrailsCompileStatic.

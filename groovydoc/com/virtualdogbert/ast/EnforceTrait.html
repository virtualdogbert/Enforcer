Package: com.virtualdogbert.ast

[Groovy] Trait EnforceTrait

    • com.virtualdogbert.ast.EnforceTrait 

    • @groovy.transform.Trait
@groovy.transform.CompileStatic
trait EnforceTrait
extends java.lang.Object

      This trait provides all the base functionality for various enforcer AST Transforms(wrapping methods, dealing with parameters, etc). It has one abstract method additionalMethodProcessing, used to apply the actual enforcer transform, and any other addons like transactionality, static compilation, or both.
